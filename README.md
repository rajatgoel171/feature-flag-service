# Feature Flag Service

A dependency-free feature flag evaluator for Node.js. It supports targeted users, attribute rules, and deterministic percentage rollouts.

## What This Demonstrates

- Product platform architecture
- Deterministic rollout logic
- Clean and testable domain code
- Practical configuration modeling

## Features

- Boolean feature flags
- User allowlists
- Attribute-based targeting
- Percentage rollout bucketing
- Deterministic evaluation results

## Run

```bash
npm test
npm run demo
```
